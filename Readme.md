# fish-git-emojis

Shortcuts to commit with Gitmoji messages

## 🚀 Install

```console
fisher add Gazorby/fish-git-emojis
```

## 🔧 Usage

| Command | Description |
|-------- | ----------- |
| gbuild  | build/chore |
| gci     | Continuous integration |
| gdoc    | Documentation |
| gfix    | Bugfix |
| gfeat    | New feature |
| gperf   | Improve performance |
| gref    | Code refactoring |
| gstyle  | Code style |
| gtest   | Test |
